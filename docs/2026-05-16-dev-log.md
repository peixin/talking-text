## 4. V1 完整闭环（用户视角）

```
/register
  ↓ 注册（无 learner）
/onboarding/learner
  ↓ name + cefr_level（含"不确定"）→ submit
/chat
  ↓ 自动创建首个 session（如果没有），列出已有 session
/chat/[sessionId]
  顶部 banner: 📕 group_name · N items   或  ✨ Free practice   ← 可点切
  中部: Tina ↔ 孩子 对话流
  底部:
    📷🎤📎  常驻三图标 → 打开抽屉
      📷: 拍照/选图 → AI 抽取 → preview/edit → 保存切 scope
      🎤: 录音 → 转写填 description → Extract
      📎: 选图 → 同 📷
    [文字/语音切换] + 输入区
  
  后台:
    每轮 anchor-scan 更新 mastery seen_count
    NULL cefr_level → 每轮静默 LLM 估算等级，3-5 轮 settle 后写回
    soft_limit 触发一次性 session 分析 → mastery used/correct/mastered_at
    Tina prompt 含 nudges：检测"今天学了"邀请拍照、话题重复时建议换

/parent
  ├ 孩子管理 (/parent/learners → /parent/learners/[id] AI 人设)
  └ 学习素材 (/parent/materials → rename / archive / delete groups)
```

**注册到产生第一个 turn：3-4 屏，≤ 2 分钟**（昨天 §12 的承诺已兑现）。
